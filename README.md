# Pokemon Recreated Game
### Made using PyGame, this game mimics the popular Pokemon! Allows you to pick a starter Pokemon, and then venture into the wild grass to randomly encounter one of three Pokemon (Zubat, Ponyta, or Jigglypuff). Take turns attacking until either you or the wild Pokemon wins.
